무럭무럭은 식물의 생육 상태를 관리하고, 갤러리 기능을 통해 식물과의 이벤트를 저장하며 그것을 커뮤니티로 공유하는 앱입니다.

무럭무럭의 특징

무럭무럭은 식물 관리, 식물 갤러리, 커뮤니티라는 3가지 특징이 있습니다

식물 관리

식물을 등록하여 물 주기를 관리할 수 있습니다.
식물에 대해 잘 모르신다면 식물 도감을 확인해 보세요.

나만의 식물 갤러리

식물의 모습을 사진으로 남기고 싶으시다구요?
식물 갤러리에 사진을 올려 보세요.

커뮤니티

공유된 사진을 식물 종류별로 보여주는 커뮤니티를 제공합니다.

무럭무럭의 주요 기능

쉬운 물 관리

오늘 물을 줬다면 물방울 버튼을 클릭하세요.
물 주기에 대한 히스토리는 달력에서 확인 할 수 있고 물 알람을 설정할 수도 있습니다.

식물 도감

식물 도감에서는 비료 정보, 생육 환경, 계절 별 물 주기를 확인할 수 있습니다.

식물 갤러리

기억하고 싶은 사진을 갤러리에 올려보세요. 무럭무럭은 사진의 날짜 별로 필터링 또한 제공합니다.

식물 별 커뮤니티

좋아요, 댓글 기능을 활용하여 다른 유저와 커뮤니케이션 하세요. 만약 맘에 드는 사진이 있다면 카톡 친구에게 공유할 수 있습니다.

무럭무럭에 사용된 기술

파이어베이스


실시간 DB를 이용해 데이터를 저장하였습니다.

저장소에 이미지를 업로드하여 커뮤니티 화면의 이미지를 url을 이용해 불러왔습니다.

인증을 활용하여 구글, 페이스북 로그인 기능을 추가하였습니다

공공 API 활용

농촌 진흥청의 실내 정원용 식물 API를 이용하여 받아온 데이터를 토대로 식물 도감을 구성하였습니다.

Glide

이미지 로드 라이브러리인 Glide를 사용하였습니다.

이외에도 expandable layout, circle imageview, material calendarview, ted permission 등의 오픈소스 라이브러리를 사용하였습니다

저는 안드로이드에 대해 더 깊게 공부하고 싶어서 부스트캠프에 입과했었습니다. 그런데 입과 한 후 이전의 저는 아무것도 모르는 백지 상태였다는 것을 깨달을 수 있었습니다. 부스트 캠프에서 공부하면서 저는 백지에 점 하나를 찍었다고 생각합니다. 앞으로도 계속 공부하면서 좋은 개발자로 성장하겠습니다.
